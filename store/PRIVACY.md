# 隐私政策 · 视频倍速控制（Video Speed Controller）

最后更新：2026-06-14

## 概述
「视频倍速控制」是一个用快捷键控制网页视频播放速度的浏览器扩展。本扩展**不收集、不存储、不传输任何个人数据**。

## 我们不收集的数据
本扩展不会收集或上传以下任何信息：个人身份信息、健康信息、财务信息、认证信息、个人通讯、位置、浏览历史、用户操作行为等。扩展不包含任何分析、追踪或广告组件，也不会向任何外部服务器发送请求。

## 本地存储的内容
扩展仅在浏览器内保存你的偏好设置（快捷键、倍速步长、一键倍速目标、各开关、以及可选的"上次速度"）：
- 配置通过 Chrome 的 `storage.sync` 保存，并在你登录 Chrome 账号时由 Chrome 在你的设备间同步。该同步由 Google/Chrome 提供，扩展开发者无法访问这些数据。
- "上次速度"等设备本地状态保存在 `storage.local`，仅存在于本机。

这些数据**只用于实现扩展自身功能**，不会被读取用于其他目的，也不会离开你的浏览器/你的 Chrome 账号。

## 权限用途
- `<all_urls>`：在任意网站上找到并控制 `<video>`/`<audio>` 的播放速度。仅读取媒体的播放速率，不读取页面其他内容。
- `storage`：保存上述偏好设置。
- `scripting` / `activeTab`：当你在弹窗中读取/设置当前页面视频速度时，向当前标签页注入一小段脚本。

## 远程代码
扩展的全部代码随安装包一起分发，不从外部加载或执行任何远程代码。
