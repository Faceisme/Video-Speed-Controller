## 详细描述（Description）

```
用快捷键随手控制任意网页视频的播放速度，无需点开播放器菜单。

【主要功能】
• 快捷键加速 / 减速 / 一键倍速 / 复位
• 快进 / 快退
• 视频左上角实时显示当前速度，可常驻
• "强制保持"：当网站把速度偷偷改回 1.0 倍时自动复位到你设定的速度
• 支持跨域 iframe 内嵌视频与预览视频
• 支持动态加载的播放器（边播边出现的视频也能控制）

【默认快捷键】
D = 加速      S = 减速
A = 一键倍速（在 1.0x 与设定倍速之间切换）
R = 恢复 1.0x
Z = 快退      X = 快进
V = 常驻 / 隐藏速度提示
（均可在弹窗中自定义；在输入框打字时快捷键自动失效）

【设置同步与备份】
• 配置通过 Chrome 账号自动跨设备同步
• 支持导出 / 导入配置文件，方便备份或迁移到其他浏览器

【隐私】
本扩展不收集、不上传任何数据。所有设置仅保存在你的浏览器本地及 Chrome 账号同步存储中。
```

## 权限说明（Permission Justifications）

提交时每项权限后台会要求填写理由，逐条对应：

**host_permissions: `<all_urls>`**
扩展需要在用户访问的任意网站上找到 `<video>` / `<audio>` 元素并设置其播放速度，因此请求对所有网站的访问权限。扩展只读取媒体元素的播放速率属性，不读取、不收集、不上传任何页面内容或浏览数据。

**`storage`**
保存用户的快捷键、倍速步长等偏好设置，并在用户登录 Chrome 账号时跨设备同步这些设置。

**`scripting`**
当用户在扩展弹窗中读取或设置当前页面视频速度时，向当前标签页注入一小段脚本以读写视频元素的 playbackRate。仅在用户主动操作弹窗时触发。

**`activeTab`**
弹窗中的速度操作只作用于用户当前激活的标签页。

**为什么需要在所有网站运行（单一用途说明 / Single Purpose）**
本扩展的唯一用途是：通过快捷键控制网页视频的播放速度。由于用户可能在任意网站观看视频，必须能在所有网站上运行才能实现该功能。

## 数据用途声明（Privacy practices 表单答案）

- 是否收集或使用用户数据？**否（不收集任何用户数据）**
- 个人身份信息 / 健康 / 财务 / 认证信息 / 个人通讯 / 位置 / 网页浏览记录 / 用户活动：**均不收集**
- 是否将数据出售给第三方？**否**
- 是否将数据用于与核心功能无关的用途？**否**
- 是否用于判断信用 / 放贷？**否**
- 远程代码（Remote code）：**不使用**（全部脚本随扩展打包，无外部加载）

> 由于本扩展不收集任何数据，通常无需隐私政策网址；但带 `<all_urls>` 宽泛权限时，审核可能要求提供一个隐私政策 URL。仓库里已附 `store/PRIVACY.md`，可把它放到 GitHub Pages / Gist 等任意公开网址后填入。
