# 视频倍速控制 · Video Speed Controller

用快捷键控制**任意网页视频**的播放速度。支持跨域 iframe 嵌入的视频、预览视频、动态加载的播放器，并带「强制保持」功能——当网站偷偷把速度改回 1.0 时自动复位到你设定的倍速。

## 安装（开发者模式加载）

1. 打开 Chrome，地址栏输入 `chrome://extensions`
2. 右上角打开「开发者模式」
3. 点「加载已解压的扩展程序」，选择本文件夹 `chrome_ext`
4. 安装后，**刷新**你要使用的视频页面（已经打开的旧标签页需要刷新一次内容脚本才会注入）

## 默认快捷键

| 按键 | 功能 |
| --- | --- |
| `D` | 加速（+步长，默认 0.25） |
| `S` | 减速（−步长） |
| `R` | 恢复 1.0x |
| `A` | 一键倍速（在 1.0x 与设定倍速如 2.0x 之间切换） |
| `Z` | 快退（默认 5 秒） |
| `X` | 快进（默认 5 秒） |
| `V` | 常驻 / 隐藏左上角速度提示 |

> 在输入框、文本域里打字时快捷键自动失效，不会误触。按住 Ctrl/⌘/Alt 时也不拦截。

## 弹窗（点工具栏图标）

- 直接点预设按钮或用 `+ / −` 设速度
- 修改步长、一键倍速目标、快进秒数
- 「记住上次速度」：新视频自动套用上次速度
- 自定义快捷键：点对应输入框后按一下你想要的键
- **导出 / 导入配置**：备份设置，或搬到其他浏览器（如 Edge）

## 配置同步与备份

- **跨设备同步**：配置存在 `chrome.storage.sync`，登录同一 Google 账号并开启 Chrome 同步后，多台电脑的设置自动共享。未登录时它退化为本机存储，照常可用。
- **导出/导入**：弹窗底部「导出配置」生成 `video-speed-config.json`；在另一台机器或另一个浏览器点「导入配置」选择该文件即可。
- 「记住的上次速度」属于设备本地状态（存 `local`、按键时防抖写入），不参与同步，避免触发 sync 的写入频率配额。

## 关于你的目标站点

- `tameikegoro.jp`、`video.dmm.co.jp` 的预览视频都是 HTML5 `<video>`，倍速直接生效。
- DMM 的视频常嵌在跨域 iframe 内，扩展用 `all_frames` 注入到每个 frame，并把顶层按键通过 `postMessage` 中继进 iframe，所以**即使没点进 iframe 也能改速**。若某个站点焦点在 iframe 里，直接在视频上点一下再按键最稳。

## 文件结构

```
manifest.json   扩展清单(MV3)
content.js      核心:媒体发现 / 倍速 / 强制保持 / 快捷键 / iframe 中继
content.css     左上角速度提示样式
popup.html/.css/.js   工具栏弹窗 UI 与设置
```

## 可选：添加图标

当前未带图标（用 Chrome 默认拼图图标，不影响功能）。如需自定义，放入 `icons/` 后在 `manifest.json` 增加：

```json
"icons": { "16": "icons/16.png", "48": "icons/48.png", "128": "icons/128.png" },
"action": { "default_icon": "icons/48.png", "default_popup": "popup.html" }
```
